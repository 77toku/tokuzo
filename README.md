## GitHub Pagesで公開する方法

1. GitHubで新しいリポジトリを作成
2. この3ファイルをリポジトリ直下へアップロード
3. `Settings` → `Pages`
4. `Deploy from a branch` を選択
5. `main` / `root` を選択して保存
